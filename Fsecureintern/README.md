

Introduction:-

(1) The project is based on Calendar public API (Nager.Date) from github. This API provides Public hoildays from more than 90 countries.
    The API URL is here for your reference https://date.nager.at/api/v2/PublicHolidays/2021/us

(2) I have created a web application that will allow the users to view public holidays in any country from the available ninety countries. 
    The project is based on Asp.net core 3.1 Mvc web application architecture.

(3) This application requires visual studio 2019 to run.


Steps:-

Steps for running the project:

(1) Download or clone the git repository to your computer.
(2) Unzip the folder.
(3) From the FsecureInternTask folder copy the folder which is Fsecureintern to the desktop.
(4) Open the Fsecureintern folder and double click on Fsecureintern.sln file which is basically a solution file for opening the project in visual studio.
(5) Once you succeed in opening the project folder in visual studio just press the run button and you go there.
(6) Provide the Country code as i.e. PL for Poland and year in YYYY (i.e. 2019 ) format.
(7) Congratulations you succeed.
